*Testado na versão mais recente do python 3*
Para executar esse programa, é necessário ter o Python 3 instalado.

Instruções caso tenha apenas o pyhton 3 instalado:
1-Instale a biblioteca "pandas"(caso ainda não tenha) com o comando:pip install pandas (pode precisar de permissão de administrador)
2-Abra um terminal dentro da pasta do programa.
3-Rode o comando:python at10.py
4-O resultado será gerado no arquivo "output.txt", dependendo de qual opção seja escolhida no programa.
OBS = O nome do arquivo deve ser colocado como input na execução do programa.
OBS² = O arquivo "completar_dados.py" possui funções implementadas em outras tarefas que foram utilizadas nessa.

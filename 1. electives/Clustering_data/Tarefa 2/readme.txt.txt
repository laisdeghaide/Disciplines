*Testado na versão mais recente do python 3*
Para executar esse programa, é necessário ter o Python 3 instalado.

Instruções caso tenha apenas o pyhton 3 instalado:
1-Instale a biblioteca "pandas"(caso ainda não tenha) com o comando:pip install pandas (pode precisar de permissão de administrador)
2-Abra um terminal dentro da pasta do programa.
3-Rode o comando:python main.py
4-O resultado será gerado no arquivo "userprofile_output.csv"

Instruções caso tenhas o pyhton 3 e o python 2 instalado:
1-Instale a biblioteca "pandas"(caso ainda não tenha) com o comando:pip3 install pandas (pode precisar de permissão de administrador)
2-Abra um terminal dentro da pasta do programa.
3-Rode o comando:python3 main.py
4-O resultado será gerado no arquivo "nomedoinput_output.csv"
